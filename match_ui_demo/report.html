3. 해석

  • 출전 균등 — 실용 모드 전부 1인 7~8게임(±1). "경기수 적은 순" 정렬만으로 공정성 견고. ✅
  • 실력 균형 — 평균 실력차 ≤0.33점, 거의 모든 경기가 ±1급수 이내 박빙. ✅
  • 밸런스 ↔ 동일급수 트레이드오프 — 동일급수는 균형을 거의 완벽(0.01)하게 맞추는 대신 다양성 양보(첫 파트너 94%→87%). 밸런스는 다양성이 더 높음. 설계 의도대로 작동.
  • 남복/여복 — 성별로 풀이 갈려 후보가 적어 실력차가 다소 큼(0.33)이지만 박빙률 99%.
  • 혼복만은 비실용 — 남35:여16에서 여자가 8코트를 채우는 순간 소진되어 11게임에서 멈춤(1인 0.9게임). 현장에선 "혼복 불가" 알림 반복. → 이 인원엔 모두 또는 남복/여복 권장.
