🎚️ 종목 비율 손잡이 — 프로토타입 검증

작성일 2026-06-15 · "모두" 모드에서 혼복/남복/여복 비율을 부드럽게 조절
문제: 모두 모드는 여자가 소수라 자연히 혼복으로 쏠려, 여복이 거의 안 생긴다(여자 절반이 여복 0회). 해법: 운영자가 대략의 목표 비율만 정하면, 시스템이 뒤처진 종목에 선택 보너스를 줘 그쪽으로 부드럽게 당긴다(강제 X). 같은 모수·난수로 손잡이 단계별 효과를 측정했다.
측정 조건: 현실 ABCD 모수 51명(남 30·여 21) · 8코트 · 3시간 · 게임 13~17분 랜덤 · preset 밸런스

해석

  • 손잡이가 분명히 작동 — 여복이 baseline 4게임 → "남녀복 많이" 16게임으로 4배. 여복 뛴 여자도 12명 → 21명 전원.
  • "골고루"만 줘도 충분 — 여복 12게임, 여자 19/21명이 여복 경험. 분위기 환기엔 이 정도가 적당.
  • 품질은 그대로 — 어느 단계든 출전 7~8 균등 · 박빙률 100% · 다양성 95~97% 유지. 비율만 바뀌고 매칭 품질 손상 없음.
  • 목표를 100% 강제하진 않음 — 목표 34:33:33을 줘도 실제는 56:28:16. 여자가 41%뿐이라 여복을 33%까지 못 끌어올린다. 이게 의도한 soft(가능한 만큼만 당김) 동작. 무리하게 코트를 놀리거나 박빙을 깨지 않는다.
  • 운영자 UI는 3단계 토글(혼복 위주 · 골고루 · 남녀복 많이)이면 충분. 지금 종목 모드 토글 옆에 하나 더.
✅ 결론: "혼복 쏠림"을 운영자가 손잡이 하나로 부드럽게 조절 가능함을 확인. 출전·균형·다양성을 해치지 않고 종목 비율만 의도대로 이동. (core.jsx에 옵션으로 구현됨 — UI 토글만 붙이면 실제 적용)
